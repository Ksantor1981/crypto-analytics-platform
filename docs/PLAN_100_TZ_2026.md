## 2. Ошибки и риски

| Тип | Описание |
|-----|----------|
| **Несоответствие ТЗ** | Coverage 29% vs 80%; ML accuracy real vs CV; нагрузка не тестировалась |
| **Архитектура** | Alembic: «миграции не сгенерированы» — противоречит «миграция добавлена»; уточнить актуальность |
| **Сбор без TG API** | `telegram_scraper` — t.me/s (HTTP), без ключей; `telethon_collector` — не использовать |
| **Scheduler** | При отсутствии TG API сбор из TG через t.me/s возможен, но риск rate-limit; Reddit — без ограничений |
| **CI** | E2E continue-on-error; 1 skipped тест |
| **Дубликаты** | `cleanup_duplicates` по full text; при вставке — по left(500); возможны квази-дубли |
